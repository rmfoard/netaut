To provision for generating netaut/hellosnap:
- sudo apt install libpng-dev
- download graphviz...gz
-   gunzip ...gz
-   tar xvf ...tar
-   cd graphviz...
-   ./configure
-   make
-   sudo make install
- clone SNAP from Github into ~/snap
-   make all
